# Projects

## Autonomous Mobile Robot for Medicine Delivery

Designed and built a 2D LiDAR-based AMR for hospital medicine delivery.

Highlights:

- ROS navigation stack
- PID motor control
- Odometry + IMU fusion
- Gazebo simulation
- Remote UI for topological navigation

Funded by **ARTPARK, IISc Bangalore**.

## AGV Navigation System

Worked on high-speed AGV navigation at **Addverb Technologies**.

Highlights:

- Implemented geometric controllers (Pure Pursuit, Stanley, Vector Pursuit)
- Reduced cross-track error during high-speed motion
- Developed cruise controller using LiDAR distance and velocity estimation
- Optimized robotics stack to run on **2 CPU cores and 1.6GB RAM**
